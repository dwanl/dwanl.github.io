reduce 函数详解

发表于 2019-12-09 | 分类于 JavaScript

reduce 方法介绍：

reduce() 方法对数组每个元素执行一个 reduce 函数，将其结果汇总为单个返回值。

reduce 方法使用场景：

1. 对数组进行求和
2. 将二维数组转换为一维数组
3. 计算数组每个元素出现的次数

filter 方法介绍：

filter() 方法创建一个新数组，其成员是通过提供函数测试的所有元素。
